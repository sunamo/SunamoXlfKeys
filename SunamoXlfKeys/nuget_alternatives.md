# NuGet Alternatives to SunamoXlfKeys

This document lists popular NuGet packages that provide similar functionality to SunamoXlfKeys.

## Overview

XLF localization key utilities

## Alternative Packages

### Microsoft.Extensions.Localization
- **NuGet**: Microsoft.Extensions.Localization
- **Purpose**: Localization framework
- **Key Features**: Resource strings, cultures

### XliffParser
- **NuGet**: XliffParser
- **Purpose**: XLIFF file parsing
- **Key Features**: Read, write XLIFF files

### Multilingual.NET
- **NuGet**: Multilingual.NET
- **Purpose**: Localization utilities
- **Key Features**: XLIFF support, translation workflows

## Comparison Notes

XliffParser for XLIFF files. Microsoft.Extensions.Localization for general localization.

## Choosing an Alternative

Consider these alternatives based on your specific needs:
- **Microsoft.Extensions.Localization**: Localization framework
- **XliffParser**: XLIFF file parsing
- **Multilingual.NET**: Localization utilities
